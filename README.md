# Haylee AI - Your personal Claude Code for Marketing

Haylee AI is a fully agentic and AI-native tool to automate managing of ads. 
The goal of this project is to simplify Meta Ads Manager as it's too complex for most people.

We provide the abstractions necessary and/or let AI handle the core work of managing and reporting + optimization of ad performance.
